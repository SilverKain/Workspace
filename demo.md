# Демонстрационный Markdown файл

Добро пожаловать в **Markdown Viewer**! Это приложение создано в стиле **Google NotebookLM**.

## Основные возможности

### 1. Загрузка файлов
- Нажмите кнопку "Загрузить файлы"
- Выберите `.md` или `.txt` файлы
- Файлы сохраняются локально в браузере

### 2. Создание проектов
- Кликните "Создать проект"
- Перетаскивайте файлы в проект
- Организуйте структуру знаний

### 3. Статистика
- Календарь показывает активность
- Счётчики открытий файлов
- История по датам

## Примеры форматирования

### Списки

**Нумерованный список:**
1. Первый пункт
2. Второй пункт
3. Третий пункт

**Маркированный список:**
- Пункт А
- Пункт Б
- Пункт В

### Код

Inline код: `const greeting = "Hello World";`

Блок кода:
```javascript
function createProject(name) {
    return {
        id: generateId(),
        name: name,
        files: [],
        expanded: true
    };
}
```
